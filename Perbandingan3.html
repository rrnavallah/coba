Perbandingan 3 :
PERANCANGAN SISTEM INFORMASI KEPEGAWAIAN BERBASIS WEBSITE DI BAGIAN KEPAGAWAIAN SDN BINAKARYA I KABUPATEN GARUT
LATAR BELAKANG

Latar belakang PERANCANGAN SISTEM INFORMASI KEPEGAWAIAN BERBASIS WEBSITE DI BAGIAN KEPAGAWAIAN SDN BINAKARYA I KABUPATEN GARUT Karena semakin dibutuhkanya system informasi untuk kinerja organisasi. Dan penataaninformasi juga harus dilakukan secara teratur,jelas,tepat an cepat serta disajikandenan bentuk laporan ntuk mendukung kegiatan operasiaonal dan pengambilan keputusan yang tepat. subbag administrasi UIN Jakarta yang memilikki tugas melakukan administrasi mutsi kepegawaian meliouti masalah kenailan pangkat jabatan, kenaikan gaji berkala, dan pension kepegawaian. Berdasarkan hasil observasi dan wawancara peuls dengan KAsubag Administrasi Kepegawaian, pengelolaan nformasi kepegawaian sudah dilakukan dengan terkomputerasi. Data data mengenai kepegawaian jsudah berada di dalam sutau data base dengan program aplikasi Sistem Kepegawaian (SIMPEG) berbasis desktop. Namun, aplikasi tersebut masih memiliki ekurangandiantaranya belum sempurna dwmpurnanya fitur untuk pembuatan Surat Keputusan kenaikan pangkatdan kenaikan gaji berkalasehingga masih dilakukan dengan menggunakan file file Exel dan Word. Elain itu juga belum terdapatnya modul administrasi pension pada aplikasi SIMPEG. Untuk masalahpensiun masihdikalkuan secra semi terkomputerasu artinya data pegawai yang mengajukan pension dinputkan ke komutermengunakan program pengelolaan data. Sebagai dokumeentasi untuk kemdaiandibuatkan Surat USulan Pensiundan Surat Pengantar yang akan ditutnjukan pada Bisro administrasi Umum dan KEpeawaian. Laporan yang dihasilkan dari system juga belum memenuhi kebutuhan manajemen kepegawaian UIN Syarif Hidayatullah Jakarta.

Sedangkan latar belakang pada SISTEM INFORMASI MANAJEMEN KEPEGAWAIAN PADA SMK X KEMIRI PURWOREJO adalah karna pada Tenaga tata usaha mempunyai tugas untuk mengolah data siswa, data pembayaran sekolah, data kepegawaian. Pengolahan data kepegawaian dilakukan untuk mengetahui data pegawai yang bekerja di sekolah. Data kepegawaian diperlukan untuk mengetahui kesejahteraan para tenaga pegawai yang bekerja disekolah. Namn, selama ini sistem pengolahan data yang dilakukan oleh petugas tata usaha dalam pengolahan data guru, siswa, dan lainya pada SMK X Kemiri masih secara manual.

RUMUSAN MASALAH

Perumusan Masalah pada PERANCANGAN SISTEM INFORMASI KEPEGAWAIAN BERBASIS WEBSITE DI BAGIAN KEPAGAWAIAN SDN BINAKARYA I KABUPATEN GARUT diantara lain

1. Bagaimana pengembanganSIMPEG yang saat iniberbasi desktop menjadi SIMPEG bebasis WB
2. Bagaimana mengembangkan SIMPEG menjadi pusat data yang terseentralisasi sebagai pelayanan kepegawaian di setiap unit kerja diUIN?
3. Bagaimana membuat laporanyang dihasilkan oleh system sesuai dengan format yang dibutuhkan SUBBAG Administrasi KEpegawaian Pusat UIN Jakarta
4. Bgaimana menyempuranakan fitur untuk pembuatan SK sehingga untuk memnuat SK kenaikan pangkat dan kenaikan gaji berkala dilakukan secara otomatis dari system?
5. Bagaimana membat modul pengolahan dan penyimpanan data pension pegawai yang sesuai dengan proses bisnis yang berjalandi Subbag Administrasi KEpegawaian Pusat UIN Jakarta?
Sedangkan rumusan masalah pada SISTEM INFORMASI MANAJEMEN KEPEGAWAIAN PADA SMK X KEMIRI PURWOREJO adalah bagaimana membuat system informasi manajemen kepegawaian agar dapat membantu mengelola dan menyimpan data pekerjaan pada SMK X KEMIRI di Purworejo dengan efektif dan efisien.

TUJUAN MASALAH

Tujuan pada PERANCANGAN SISTEM INFORMASI KEPEGAWAIAN BERBASIS WEBSITE DI BAGIAN KEPAGAWAIAN SDN BINAKARYA I KABUPATEN GARUT adalah

1. Menganalisis system yang ada pada Subbagian Administrasi KEpegawaaian Pusat UIN Jakarta
2. Mengembangkan aplikasi kepegawaian yang masih berbasis desktop menjadi berbasis web
3. Merancang aplikasi kepegawaianyang dapat menjunjung aktivias administrasi kepegwaian
4. Menguji aplikasi kepegawaian yang baru dengan metodr blockbox testing.
Sedangkan tujuan dari pada SISTEM INFORMASI MANAJEMEN KEPEGAWAIAN PADA SMK X KEMIRI PURWOREJO Merancang sisitem informasi kepegawaian berbasis web pada SMK X KEMIRI di Purworejo dengan menggunakan Bahasa pemrograman HTML, CSS, JavaScript dan MySql agara petugas tata usaha menyimpan data-data tersebut dalam arsip-arsip yang tertata rapi tetapi membutuhkan tenaga dan waktu ekstra dalam pencarian data.

BATASAN MASALAH

Batasan Masalah pada PERANCANGAN SISTEM INFORMASI KEPEGAWAIAN BERBASIS WEBSITE DI BAGIAN KEPAGAWAIAN SDN BINAKARYA I KABUPATEN GARUT sebagai berikut

1. Analisis dan perancangan darabease yang akan menamoun kesekuruhan data pegawai UIN Syarif Hidayatullah Jakarta menggunakan MySql versi 5.0.67 sebagai tools
2. Analisis dan perancanganaplikasi berbasis web denan menggunakan Bahasa pemrograman HTML dan PHP versi 5.2.6
3. Permasalahan penelitian dibatasi hanya untukpegawai administrasi golongan III/d kebawah
4. Pengembangan modul data pegawai, data kenaikapangkat data kenaikan gaji berkala, dan data pensiun.
5. Perancangan penyajian format laporan terkait kepegawaian yang sesuai dengankebutuhan manajerial
6. Perancangan dan pembuatan fitur Surat Keputusan (SK) kenaikan pangkat da Surat Keputusan(SK) kenaikan gaji berkala pegawai.
7. Dalam melakukan perancanganaplikasi,penulis menggunakan Diah=gram Alir, DFD, serta STD dan ERD sebagai permodelan database.
8. Pengguna system ini nantinya adalah pegawai, bagian kepegawaian unit kerja sebaai admin unit kerja, subbag administrasi kepegawaian pusat sebagai admn osat,kepala biro kepegawaian dan kepala bagian kepegawaian sebagai kepala, pihak PUSKOM, dan Administrator system.
Sedangkan Batasan masalah pada SISTEM INFORMASI MANAJEMEN KEPEGAWAIAN PADA SMK X KEMIRI PURWOREJO adalah
1. Penelitian hanya mencakup dan berfokus pada pegawai SMK X KEMIRI Purworejo.
2. Data yang diolah hanya meliputi golongan pegawai, gaji pegawai.
3. Menggunakan diagram UML.
4. Pengembangan system dengan menggunakan Bahasa pemrograman HTML, CSS, JavaScript dan MySql untuk merancang database.
